Files Structure

Our template consists of :

Root files:

  • index.html
  • favicon.ico
  • config.rb

Folders: 

  • vendor (webgl scripts)
  • sass (sass files according to each sections)
  • js (js files tha we are using in template)
  • img (images folder)
  • fonts (fonts folder)
  • css (final generated css file)

The entire template is built on a grid bootstrap using basic styles and scripts. Separately we want you to pay attention to the stylistic structure of the layout. Using the compass has allowed us to develop a flexible structure style files. To configure settings of project in config.rb in root of archive. In sestion folder you will find a breakdown of style files by the same name sections of the layout. Also all animation is done by Animated.css. 

Its very simple to use:

For ex. all section titles has the same animation effect flipInX.
In template you could find somthing like this:

<h2 class="text-center revealOnScroll"  data-animation="flipInX" data-timeout="100" >Title</h2>

All animation effects you could find here.

Animated Sales Diagramm section

Animated Sales Diagramm section is set providing information in the form of charts and graphs.

In main.js you could find code to animate circle diagramm:

$window.on('scroll resize', function() {
   $('#diagrams').find('.diagram_circles').each(function(n,e){
       var window_height = $window.height();
       var window_top_position = $window.scrollTop();
       var window_bottom_position = (window_top_position + window_height);
       var $el = $(e);
       var el_height = $el.outerHeight();
       var el_top_position = $el.offset().top;
       var el_bottom_position = (el_top_position + el_height);
       if ( 0 == $el.data('run') ) {
         if ( (el_bottom_position >= window_top_position) 
               && (el_top_position <= window_bottom_position) ) {
           $el.circliful({
               percent: $el.data('percent')
           });
           $el.data('run', '1');
         }
       }
   });
});
Animation Graphs is realized via pure css.
